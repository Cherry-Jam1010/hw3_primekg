📖 项目介绍

本项目基于 PrimeKG（Precision Medicine Knowledge Graph） 构建一个面向心理学与精神健康领域的知识图谱分析系统。 PrimeKG 整合了 20 个高质量生物医学资源，描述了 17,080 种疾病和超过 405 万条关系。

系统核心功能包括：
1. 子图提取与分析：从 PrimeKG 中提取精神障碍子图，分析其本体结构、节点类型分布、关系类型分布
2. 知识图谱嵌入：训练 TransE 与 RotatE 模型，通过 MRR、Hits@K 等指标评估链接预测性能
3. 疾病大类预测：基于图嵌入表示对疾病节点进行分类，探索图谱结构反映的疾病相似性
4. LLM 幻觉检测：评估 LLM 在精神障碍知识上的幻觉倾向，区分真实与伪造三元组
5. RAG 增强问答：对比纯 LLM 与 RAG 在回答准确性、证据可追溯性上的差异
